# Copywork

This folder is where you write code from memory.

## How it works

1. Read the section in `vault/sections/`
2. Close it
3. Create a new `.py` file here named after the section (e.g., `01_config.py`)
4. Write the code from memory
5. Diff against the original: `diff copywork/01_config.py ../../nanochat/gpt.py`
6. Note what you missed — those are the gaps in understanding

## Files

Each file corresponds to a section. Don't look at the original while writing.
When done, compare and annotate your mistakes as comments.
